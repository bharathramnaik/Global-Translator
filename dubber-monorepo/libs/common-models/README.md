# Common Models

This library contains shared DTOs and models used across the Dubber platform.

## Usage

Add this as a dependency in your service's `pom.xml` or import the models directly.

## Models

### Job Status
Enum for job status tracking across services.

### Job DTO
Standard Data Transfer Object for job information.
